# Sistema de Autenticación con Sesiones en PHP

Aplicación web que permite a los usuarios iniciar sesión, acceder a una página protegida y cerrar sesión de forma segura. 

## Características

- ✅ Sistema de login con validación de credenciales
- ✅ Manejo de sesiones PHP
- ✅ Protección de páginas mediante verificación de sesión
- ✅ Página de bienvenida personalizada
- ✅ Cierre de sesión funcional
- ✅ Página de acceso denegado
- ✅ Mensajes de error informativos
